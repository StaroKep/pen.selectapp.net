# Install Docker on Ubuntu Using Default Repositories
* `sudo apt-get update`
* `sudo apt-get remove docker docker-engine docker.io`
* `sudo apt install docker.io`
* `sudo systemctl start docker`
* `sudo systemctl enable docker`
* `docker --version`
